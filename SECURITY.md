# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities privately by emailing joeurcino@proton.me.

Include:

- A clear description of the issue.
- Steps to reproduce or a proof of concept.
- The affected OpenLuaTools version or commit.
- Any relevant logs, screenshots, or archive samples.

Do not open a public issue for security vulnerabilities. Public issues are fine
for normal bugs, feature requests, and feedback.

## Supported Versions

Security fixes are targeted at the latest OpenLuaTools release.
